# L'Espoir asbl — Site Web

Client: L'Espoir asbl  
Built by: lacønis  
Stack: HTML + Tailwind CSS (CDN) + Google Fonts  
Deploy: Vercel (static)

## Deploy

```bash
git init
git add .
git commit -m "init: homepage L'Espoir asbl"
git remote add origin YOUR_REPO_URL
git push -u origin main
```
Then connect the repo to Vercel — it will auto-detect as a static site.
