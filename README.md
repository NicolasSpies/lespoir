# L'Espoir asbl

Static site for [L'Espoir asbl](https://asbl-espoir.be) · aide à la jeunesse à Huy depuis 1985.
Built with **Astro 5** (static output) + **Tailwind CSS 3**. Self-hosted on Hostinger.

Client: L'Espoir asbl · Built by [lacønis](https://laconis.be).

## Stack

- **Astro 5.x** · static site generator
- **Tailwind CSS 3.4** · compiled to `public/tailwind.css`
- **Self-hosted fonts** · Inter Variable, Material Symbols (subset), Pally
- **Vanilla JS animations** · `public/animations.js` + `vanilla-tilt.min.js`
- **Zero external requests, zero cookies, zero tracking**

## Quick start

```bash
npm install
npm run dev        # local dev server (http://localhost:4321)
npm run build      # production build → dist/
npm run preview    # preview the dist/ build locally
```

The `build` script runs `tailwindcss` first (regenerates `public/tailwind.css`)
then `astro build` to produce the static output in `dist/`.

## Deploy to Hostinger

Upload the contents of `dist/` to the Hostinger webroot.

```bash
npm run build
# Then rsync/sftp dist/* to Hostinger
```

The `.htaccess` ships with `dist/` automatically (lives in `public/`) and handles
HTTPS, www → non-www, `.html` extension hiding, compression, caching, security
headers, and the custom 404.

## Architecture

```
src/
├── layouts/BaseLayout.astro      # <head> + nav + drawer + footer + scripts
├── components/
│   ├── Nav.astro                 # Desktop nav + hamburger
│   ├── MobileDrawer.astro        # Sliding drawer + hamburger handler
│   └── Footer.astro              # Site footer
└── pages/
    ├── index.astro               # → /
    ├── apropos.astro             # → /apropos
    ├── srg.astro / sase.astro / sapa.astro
    ├── actualites.astro / article.astro
    ├── emploi.astro / don.astro / contact.astro
    ├── mentions-legales.astro / politique-confidentialite.astro
    └── 404.astro

public/                           # static assets (copied to dist/ as-is)
├── *.webp, *.jpg, *.svg          # images and logos
├── fonts/                        # Inter, Material Symbols, Pally
├── tailwind.css                  # compiled · regenerated by build:css
├── fonts.css                     # @font-face declarations
├── animations.css                # shared animations + page transitions
├── animations.js                 # observers, counters, parallax, etc.
├── vanilla-tilt.min.js           # 3D card tilt on hover
├── llms.txt                      # AI crawler instructions
├── robots.txt
├── sitemap.xml                   # also auto-generated by @astrojs/sitemap
└── .htaccess                     # Apache config

astro.config.mjs                  # Astro · static output + sitemap integration
tailwind.config.cjs               # Tailwind 3 config
input.css                         # Tailwind entrypoint
```
